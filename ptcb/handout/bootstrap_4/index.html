Introduction to Bootstrap

There are 3 languages we must be familiar with when creating a website: HTML, CSS, JavaScript.

  • HTML: a markup language to structure the content of our website

  • CSS: a style language to decorate our HTML content to look better

  • JavaScript: a scripting language to help us add and handle complex operations in our content

Bootstrap is a collection of reusable codes written in those languages, which gives us a good starting point for creating websites.

Bootstrap also excels at making our websites responsive and it provides lots of builtin CSS class that reset and beautify the default style of many modern browsers as well.

Why Bootstrap? What is Bootstrap?

Bootstrap is free and it streamlines our development process by

  • Supporting mobile responsive design in its core.
  • Providing consistent style over different browsers and mobile devices.
  • Providing list of components and utilities to enrich our website.

By using it, we can reduce the number of custom CSS we have to write.

Bootstrap explained in video

It Makes Responsive websites

Responsive website design is the approach of making web content to relayout according to the size of screen (from Low resolution to HD monitor) without sacrificing all user experiences (UX).

Responsive frameworks, like Bootstrap, use a variety of techniques, such as grid systems and flexible images (more on these below).

It Works Like A Grid

The grid system in Bootstrap separates web content and elements in up to 12 columns. Designers and developers can lay out their content in any combination of these columns for various screen sizes.

For example, on a large screen, there are 3 columns of equal size placed alongside one another.
If screen size decreases, Bootstrap’s predefined rules will make the columns start to appear one after the other. Therefore, the web content is displayed in a readable manner without worrying about customzing it ourselves.

It Has Responsive Images

Bootstrap also includes code that allows images to be automatically resized when screen size changes, by simply adding the .img-responsive class to your images. The predefined CSS rules take care of the rest. It can even change the shape of your images automatically by adding another class to them, like .img-circle and .img-rounded.

Compatibility

Bootstrap is compatible with all modern browsers as well as Internet Explorer. Plugins like HTML5Shiv and Respond.js come as part of Bootstrap’s default template. These help in porting HTML5 elements into older non-HTML5 browsers.

Documentation

Another advantage of Bootstrap is that it provides an impressive documentation, examples and demos.
In this way, we can save our time to build the website as we do not need to explore everything before knowing how to use it. Everything is in the documentation!

Open source

One of the main reasons of using Bootstrap is the fact that it’s an open source project, hosted on Github. You don’t have to deal with purchasing and licensing issues and it gives you the freedom to change it how you want it.

Installation

There are several ways to install Bootstrap and in this unit, we are going to introduce you to the CDN and Direct Download approach.

CDN

The easiest way to try Bootstrap is to include it from CDN - Content Delivery Network. CDN is a globally distributed network of proxy servers deployed in multiple data centers and it is dedicated to deliver static content to the users in a high performant way.

Direct Download

Last but not least, you can always go to Bootstrap website and download the bootstrap library directly. This method is the most straightforward among all of the methods. One disadvantage of downloading directly is that you have to manage the version and dependency of this library manually afterwards.
